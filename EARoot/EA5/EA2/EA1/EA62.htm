Linux kernel<br/>============<br/><br/>There are several guides for kernel developers and users. These guides can<br/>be rendered in a number of formats, like HTML and PDF. Please read<br/>Documentation/admin-guide/README.rst first.<br/><br/>In order to build the documentation, use ``make htmldocs`` or<br/>``make pdfdocs``.  The formatted documentation can also be read online at:<br/><br/>    https://www.kernel.org/doc/html/latest/<br/><br/>There are various text files in the Documentation/ subdirectory,<br/>several of them using the Restructured Text markup notation.<br/><br/>Please read the Documentation/process/changes.rst file, as it contains the<br/>requirements for building and running the kernel, and information about<br/>the problems which may result by upgrading your kernel.<br/>
